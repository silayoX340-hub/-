- 🤖 **Auto AI Chat** – GPT & Gemini powered replies

- 🖼️ **AI Image Gen** – Create images with AI commands
- 🔞 **NSFW Commands** – Optional 18+ content
- 🛡️ **Group Management** – Kick, add, promote, demote, mute, lock
